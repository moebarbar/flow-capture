**STEP 1: Prepare Extension Package**

## STEP-BY-STEP LAUNCH:

**STEP 1: Choose Hosting Platform**

Recommended options (in order):

1. **Vercel** (best for frontend + serverless backend):
   - Free plan available
   - Easy deployment
   - Automatic HTTPS
   - Good for Next.js/React
   - Steps:
     - Sign up at vercel.com
     - Connect GitHub repository
     - Configure environment variables
     - Deploy (automatic)
     - Add custom domain

2. **Netlify** (alternative to Vercel):
   - Similar features
   - Great for static sites
   - Serverless functions
   - Steps same as Vercel

3. **Railway** (best if you need full backend):
   - Supports databases
   - Supports Node.js backend
   - Easy PostgreSQL setup
   - Steps:
     - Sign up at railway.app
     - Create new project
     - Connect GitHub repo
     - Add PostgreSQL database
     - Deploy

4. **AWS/Google Cloud** (for enterprise scale):
   - Most powerful but complex
   - Use if you expect high traffic
   - Requires more setup

**STEP 2: Deploy Backend**

1. Set up database:
   - PostgreSQL (recommended)
   - Create tables (run migration scripts)
   - Seed initial data if needed

2. Set environment variables:
DATABASE_URL=postgresql://...
JWT_SECRET=your_secret_key
STRIPE_SECRET_KEY=sk_test_...
STRIPE_PUBLISHABLE_KEY=pk_test_...
EMAIL_API_KEY=sendgrid_or_ses_key
AWS_S3_BUCKET=your-bucket-name
AWS_ACCESS_KEY=...
AWS_SECRET_KEY=...
API_BASE_URL=https://api.yourapp.com
FRONTEND_URL=https://yourapp.com

3. Deploy backend:
   - Push code to GitHub
   - Platform auto-deploys
   - Verify API is accessible
   - Test endpoints with Postman

**STEP 3: Deploy Frontend**

1. Build production version:
```bash
   npm run build
```

2. Set environment variables:
REACT_APP_API_URL=https://api.yourapp.com
REACT_APP_STRIPE_PUBLISHABLE_KEY=pk_test_...
REACT_APP_GOOGLE_ANALYTICS=UA-...

3. Deploy:
   - Push to GitHub
   - Platform auto-deploys
   - Verify site is live

4. Configure custom domain:
   - Buy domain (GoDaddy, Namecheap, Google Domains)
   - Add DNS records:
     - A record: @ → your hosting IP
     - CNAME record: www → your-app.vercel.app
   - Wait for DNS propagation (can take 24-48 hours)
   - Enable HTTPS (automatic on Vercel/Netlify)

**STEP 4: Final Testing on Production**

Test everything again on live site:
