   - DELETE /api/flows/:id → Deletes flow

3. Collection Endpoints:
   - GET /api/collections → Returns all collections
   - POST /api/collections → Creates new collection
   - PUT /api/collections/:id → Updates collection
   - DELETE /api/collections/:id → Deletes collection

4. Upload Endpoint:
   - POST /api/upload/screenshot → Accepts image file
   - Uploads to S3/R2/storage
   - Returns public URL
   - Validates file type (only images)
   - Validates file size (max 5MB)

5. Step Endpoints:
   - POST /api/flows/:id/steps → Creates step with screenshot URL
   - PUT /api/steps/:id → Updates step
   - DELETE /api/steps/:id → Deletes step
   - POST /api/flows/:id/steps/reorder → Reorders steps
